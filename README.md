# Palindrome CLI

Simple TypeScript CLI to check whether a word or phrase is a palindrome.

## Requirements

- Node.js (14+)
- npm

## Install

```bash
npm install
```

## Build

```bash
npm run build
```

## Run

- With an argument:

```bash
npm start -- "A man, a plan, a canal: Panama"
```

- Interactive (no arg):

```bash
npm run dev
# then type input and press Ctrl+D
```

## Test

Build and run the simple test runner:

```bash
npm run build
npm test
```
